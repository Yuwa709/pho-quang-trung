# Phở Quang Trung

The website for **Phở Quang Trung** — Northern Vietnamese food in Westminster, CA.
Rooted in Hanoi, perfected in Orange County: over 35 years of American Wagyu phở,
charcoal-grilled specialties, and family-style Northern Vietnamese feasts.

## Pages

| File           | Page                                        |
| -------------- | ------------------------------------------- |
| `index.html`   | Home                                        |
| `menu.html`    | Menu                                        |
| `about.html`   | About / our story                           |
| `visit.html`   | Visit us (hours, location, directions)      |
| `privacy.html` | Privacy policy                              |

## Tech

Static HTML site styled with [Tailwind CSS](https://tailwindcss.com/) (via CDN).
No build step — the HTML files are the site.

## Run locally

```bash
node serve.mjs   # serves the project at http://localhost:3000
```

## Project layout
